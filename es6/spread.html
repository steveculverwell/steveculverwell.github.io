The Spread Operator

Takes every item from an iterable and apply it to a new containing array

const marathons = ['london', 'loch ness', 'athens'];
const triathlons = ['london', 'blenheim', 'outlaw'];

const races = [...marathons, ...triathlons];

// console.log(races);
// (6) ["london", "loch ness", "athens", "london", "blenheim", "outlaw"]

Using Spread to copy an existing array

To copy an array rather than reference it. Meaning you can change an index of the array and not affect the original

const completedRaces = [...races];
completedRaces[0]= "Brighton";

// console.log(completedRaces);
// (6) ["Brighton", "loch ness", "athens", "london", "blenheim", "outlaw"]
// console.log(races);
// (6) ["london", "loch ness", "athens", "london", "blenheim", "outlaw"]
